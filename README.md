注：
1. 中英文的分割符是“ - ”，注意'-'左右两边都要有空格。这是为了防止一些单词自己带‘-’。
2. 词库需要自己做一下，现在大模型很好用了，我个人是一边背百词斩一边写英文，然后指定一下示例格式，直接丢进文心一言里面了。
3. 源码很简单，py入门级别，没有任何框架，很好懂，想加什么功能自己改。
4. 真的很好用，你会爱上的。
